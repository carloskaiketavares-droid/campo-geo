SOP resumido — como gerar o KMZ
1. Google Earth (web/desktop) → criar polígono da área de atuação.
2. Salvar como .kmz.
3. Guardar o original em 06_CAMPO_GEO/KMZ_FONTES/{projeto}/.
4. Voltar aqui → selecionar projeto → carregar arquivo → publicar.
5. O operador vê o polígono ao abrir o projeto no app.
Detalhes: veja MANUAL-KMZ.md na pasta do projeto.
